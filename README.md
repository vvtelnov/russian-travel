# Проект: Путешествие по России

### Обзор
Это проект о путешествии по России. 

В Фигме размещен макет, на основе которого была осуществлена адаптивная верска для самых распространённых разрешениях экранов.

**Технологии**
* HTML5
* CSS
* BEM-nested
* Flexbox
* Grid
